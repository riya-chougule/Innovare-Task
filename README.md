# Innovare-Task

## Setup

1. Create virtual environment:

```bash
   python3 -m venv myenv
   source myenv/bin/activate
```
2. Install dependencies:
```bash
pip install -r requirements.txt 
```

3. Place CSV files in data/ folder.

4. Run pipeline:   
```bash
python src/main.py
```
5. Logs are saved in logs/app.log
